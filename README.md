# BitMiner
## 코인을 채굴하고, 장비를 업그레이드하고, 돈을 모아보세요! 

# 설정
의존성 모듈들을 설치한 후 `start` 모드로 실행시킵니다.
```bash
yarn # 또는 (npm install)
yarn run start # 또는 (npm run start), (npm start)
```
